# Heroes API with WebFlux
## Stack
- Java 11
- Spring WebFlux
- Reactor
- AWS DynamoDB (Local Installation)
- JUnit
- Sl4j
- Lombok
- Swagger

## Run DynamoDB
```shell
java -Djava.library.path=./DynamoDBLocal_lib -jar DynamoDBLocal.jar -sharedDb
```
